PO Tool
версия от 9 января 2012
~~~~~~~~~~~~~~~~~~~~~~~

Утилита для обновления и форматирования исходных языковых файлов модуля Gettext.

Для работы требуется .NET Framework 2.0 - в "семерке" он предустановлен, остальные могут скачать у "мелкомягких", либо у меня:
  http://www.microsoft.com/downloads/ru-ru/details.aspx?FamilyID=5B2C0358-915B-4EB5-9B1D-10E506DA9D0F
  http://www.fallenphoenix.ru.vu/files/System/MS%20.NET%20Framework%20v2.0%20SP2%20(x86).exe_
  http://www.fallenphoenix.ru.vu/files/System/MS%20.NET%20Framework%20v2.0%20SP2%20(x64).exe_
  
Работает и на линухах под Mono - не знаю уж, какая нужна версия, но у меня на Ubuntu Lucid под Mono 2.10.5 вполне идет. Интерфейс там, правда, тупит как обычно: WinForms все-таки.
  http://www.mono-project.com
  http://badgerports.org

Мануал по использованию писать не буду: если сами не разберетесь, то она вам просто не нужна :P
